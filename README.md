# 🎓 Student Management Dashboard

A responsive Student Management Dashboard built using HTML, CSS, and JavaScript. This project helps users manage student records efficiently with Create, Read, Update, Delete (CRUD) operations and Local Storage support.

## 🚀 Features

- ➕ Add new student records
- ✏️ Edit existing student details
- 🗑️ Delete student records
- 🔍 Search students by Register Number, Name, Email, or Course
- 💾 Data stored using Browser Local Storage
- 📱 Responsive and user-friendly interface
- 🌙 Modern dark-themed dashboard

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage

## ⚙️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Start adding and managing student records.
